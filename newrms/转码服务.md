### 转码启动
```
1. 启动soffice
nohup soffice -headless -accept="socket,host=127.0.0.1,port=8100;urp;" -nofirststartwizard &

2. 启动tomcat
cd /app/tomcat/bin
./startup.sh

3. 启动zookeeper
cd /app/zookeeper/bin
./zkServer.sh start

4. 启动001，转码服务
cd /app/output/001
nohup java -jar FileTransEncode.jar &

5. 启动002，转码负载
cd /app/output/002
nohup java -jar FileTransCache.jar > /dev/null 2>&1 &
```
